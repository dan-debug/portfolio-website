Hello World!

Hello and welcome to the exciting start of a new adventure for me! Thank you for taking the time to view my current portfolio projects as I continue to challenge myself by adding new and exciting features to this website!

The Concept

The idea is that this program itself will serve as a working portfolio and continue to grow as i do. Currently this website opperates on Flask, and uses he bootstrap 4 framework for styling with a few custom CSS features.The code for this project and all others within is available via my GitHub

The Back-end

This website is current being hosted on Google Compute Engine running Debian Linux and an Apache Server with SqlLite. This hosting option is optimal as it allows for secure remote access for updates and database management. For the project to have make use of thing such as the Flask Module shared hosting was not an option.

The Front-end

The data structure for this project overall implements a "package design" to allow for clean imports and overall workflow. This project also makes use of the latest Bootstrap 4 Framework for styling and dynamic sizing capabilities!
